#### Distribution
```bash
ls /etc/*-release
cat /etc/os-release
cat /etc/issue
uname -a
cat /proc/version
```

#### Interesting files
```bash
cat /etc/passwd
cat /etc/group
cat /etc/shadow
cat /etc/hosts
ls -lh /var/mail/
ls -lh /usr/bin/
ls -lh /sbin/
```

#### Mounted File Systems
```bash
df -h
mount
cat /etc/fstab
lsblk
```

#### Loaded Kernel Modules
```c
lsmod
/sbin/modinfo <libata> // Change for any library from lsmod command
```

#### Loaded PCI and USB Devices
```bash
lspci
lsusb
```

#### CPU Info
```bash
cat /proc/cpuinfo
cat /proc/meminfo `Memory`
lshw `Hardware Info`
dmesg -T
```
